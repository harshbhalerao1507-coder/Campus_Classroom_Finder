About This System

Reference document for the Campus Classroom Finder prototype.

Project Background

Human Computer Interaction — Evaluation Project

This prototype was developed as part of a Human Computer Interaction evaluation project. The goal is to test whether students can easily locate campus facilities using a simple, self-explanatory search interface — without requiring a map, signage, or assistance from staff.

Design Objective

Zero Learning Curve

The interface is intentionally stripped to its minimal functional core. A student encountering it for the first time should be able to retrieve a result within 10 seconds.

Current Scope

Prototype Dataset

This version uses a static local dataset of 7 locations. A production deployment would integrate with the college's ERP or room-booking system to cover all facilities in real time.
